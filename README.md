# Customer Churn Prediction using Machine Learning

## Project Overview
This project predicts whether a customer is likely to leave a telecom company using a Machine Learning classification model.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Google Colab

## Machine Learning Workflow
- Data preprocessing
- Label encoding
- Train/Test Split
- Random Forest Classifier
- Model Evaluation
- Feature Importance Visualization

## Results
- Model: Random Forest Classifier
- Accuracy: 79%
